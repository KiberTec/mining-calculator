Personal Access Token (PAT) — нужен один раз
✨ Один раз настроил — работает везде. После того как ты вставишь PAT здесь, он автоматически зашифруется паролем входа и сохранится в репо. На телефоне и у партнёра ничего настраивать не надо — открыли сайт, ввели пароль → PAT подтянется сам, и можно сразу добавлять записи.
Как получить токен:
  1. Открой github.com/settings/tokens (Fine-grained)
  2. «Generate new token» → имя любое, expiration хоть No expiration
  3. Repository access → Only select repositories → KiberTec/mining-calculator
  4. Permissions → Repository permissions → Contents: Read and write
  5. «Generate» → скопируй токен (github_pat_...) → вставь сюда → Сохранить
Токен живёт в твоём браузере (localStorage) и в зашифрованном виде в репо (data/auth.json). Расшифровать его можно только зная пароль входа.
